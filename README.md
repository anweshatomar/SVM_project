# SVM_project
Analyzing the Indian Stock Market using Support Vector Machine.


## BREIF DESCRIPTION:

Built a software using  Support Vector Machine to analyse the Indian Stock Market.

Collected and cleaned the dataset consisting of five companies from three sectors.

Integrated PyQt to generate a Graphic User Interface.


## DESCRIPTION OF THE FILES:

* SVMModel.py: Consists of the main code for modeling and analysing the data.
* .py: The rest of the files are for generating a GUI.
* .csv: The dataset for the different companies that were used for analysis.
